# Weather Dashboard – Excel Project

This project demonstrates the use of Microsoft Excel to analyse and visualise weather data through an interactive dashboard.

## What this project shows
- Data cleaning and preparation in Excel
- Use of Pivot Tables and Pivot Charts
- Interactive filtering using slicers
- Clear presentation of daily and monthly weather trends

## Files in this folder
- Weather_Dashboard.xlsx – Interactive Excel dashboard
- Weather_Report.pdf – Short report explaining the dashboard design and insights
- /screenshots – Images showing the dashboard views

## Purpose
The goal of this project was to apply foundational data analysis and reporting techniques using Excel in a structured and user-friendly way.
