# UO Information Technology Fundamentals

This repository showcases selected practical work completed as part of my university studies in Information Technology Fundamentals.

The focus of this subject was understanding core IT concepts, applying technology solutions to business-style problems, and documenting technical decisions clearly for non-technical audiences.

## Key Skills Demonstrated
- Data analysis and reporting using Microsoft Excel
- Building interactive dashboards with Pivot Tables and slicers
- Basic web development using HTML, CSS, and JavaScript
- Translating business requirements into technical solutions
- Clear documentation and attention to detail

## Tools Used
- Microsoft Excel
- HTML, CSS, JavaScript
- Basic UML for system visualisation

## Projects Included
### Weather Dashboard (Excel)
An interactive Excel dashboard allowing users to explore daily and monthly weather data using Pivot Tables, slicers, formulas, and charts.

### Web Fundamentals Project
A basic website built using HTML, CSS, and JavaScript to demonstrate foundational web development concepts and simple interactivity.

## Notes
This repository contains project-style summaries and selected examples to demonstrate foundational skills. It does not include full assignment briefs or marking rubrics.
